# Senfy AI — Business Intelligence SaaS Platform

Senfy AI is a production-ready Business Intelligence platform that unifies financial analytics with sentiment intelligence to drive data-driven business decisions.

## Features

- **Production-Level Auth**: Secure signup/login with JWT and OTP verification.
- **AI-Powered Analytics**: Financial and sentiment analysis using Gemini and OpenAI.
- **Correlation Engine**: Data-backed insights explaining the "Why" behind business trends.
- **Marketing Intelligence**: Advanced module for market trends, competitor analysis, and demographic insights.
- **Professional Reporting**: Generate business-grade PDF reports with watermarks and corporate formatting.
- **Dynamic Dashboards**: Real-time interactive charts and data visualization.

## Tech Stack

- **Frontend**: React, TypeScript, Vite, Tailwind CSS, Shadcn UI, Recharts, Framer Motion.
- **Backend**: Node.js, Express, MongoDB, Mongoose, JWT, NodeMailer.
- **AI Integrations**: Google Gemini API, OpenAI API, SerpAPI.

## Getting Started

### Prerequisites

- Node.js (v18+)
- MongoDB connection string
- API Keys for Gemini, OpenAI, and SerpAPI

### Installation

1. Clone the repository:
   ```sh
   git clone <repo-url>
   cd senfy-ai
   ```

2. Install dependencies:
   ```sh
   npm install
   ```

3. Configure environment variables:
   Create a `.env` file in the root directory (see `.env.example`).

4. Start the development server:
   ```sh
   npm run dev
   ```
